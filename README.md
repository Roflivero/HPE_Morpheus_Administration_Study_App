# HPE Morpheus Administration Study App

Static study quiz for HPE Morpheus Enterprise Software Administration.

## Run Locally

Open `index.html` in a browser, or serve the directory with any static web server.

## Deploy On Cloudflare Pages

Use the GitHub repository as the Pages source.

- Framework preset: `None`
- Build command: leave empty
- Build output directory: `/`

The root `index.html` redirects to `quiz-app/`, where the application files live.

## Question Bank

The quiz bank contains 149 questions across 13 modules. Questions include a `sourcePage` field and page references in explanations so they can be checked against the course PDF.
